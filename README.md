### 「UKEditorのドキュメント」

#### はじめに

1. UKEditorとは、作者Takayuki Kamiyama(上山　孝幸)がチーム「箱入り娘」として、個人で開発しているテキストエディタです。
2. テキスト形式を基本としています。テキスト形式よりちょっとすてきなテキストを使えるように開発しています。ライセンスはBSDライセンスですので、ご自宅で使用されるときは料金を気にすることはありません。無償です。改変も、無保証だと明記すれば自由です。
3. サポートは有償です。ないとは思いますが、商用でどうしても使用したいということであれば、その都度メールにて、karuma.reason@gmail.comまで。メールには、「件名：UKEditorを商用で使いたいのでサポートをお願いしたい。」といれてください。テキストエディタを作ることが本業ではないので、そのために費やす時間と経費だけは収入にしたい、いわゆる有償ボランティアです。
4. GitHubでスポンサーを募集しています。開発を支援してくだされば、よりよく効率的に開発ができます。どうぞ、ご検討くださいませ。

#### 困ったらQ&A


はじめに何度も断っておきますがフリーソフトなので、時間をみつけてはバグをつぶすことになります。すぐに修正するのであれば、上記でのべたようにサポート代金をいただきます。直した部分を公開しないことはできません。すべての人が、同じように使えるのが理想ですからあせらず治るまで待ってください。

1. エディタの機能で動かない項目がある。
まず、動かない項目名を→次にどのように動いてほしいか、を報告してくださればできるかぎり対応します。

2. パソコンがクラッシュした。
再起動できれば、２度同じことはしないでください。ただし、事象を知りたいのでクラッシュに至った経緯を詳しく教えてください。たとえば、「保存後にコンパイラを起動したらとつぜん青い画面にいった」など。

3. テキストにこんな機能あったけ？
おそらく、ほかで実装されている機能も多々あるでしょう。私も最近まで、このヘルプアプリを震災をうけた会社からいただくまでは気分がのらなかったんですが、やる気をだしました。日々、研究しているので報告してくださればそのうち実装します。報告時、「１．名前 ２．機能の説明 ３．必要性」として説明してください。喜んでお待ちしています。

#### どうしても、ソースがほしいとき

1.GitHubのtakkiiアカウント内レポジトリに公開します。改変は、フォークし履歴がたどれるようにしてください。

2.テキスト形式には実装できる限界があると思います。できるかできないかより、とりあえず作る段取りをおいますが何かしらの理由があり実装できない場合、有志を募り共に開発してくれるようにソースはオープンソースであり続けます。
