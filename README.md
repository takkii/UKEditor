#### はじめに

```markdown
1. UKEditorとは、「箱入り娘」プロジェクトとして、個人で開発しているテキストエディタです。

2. テキスト形式を基本としています。テキスト形式よりちょっとすてきなテキストを使えるように開発しています。
ライセンスはBSDライセンスですので、ご自宅で使用されるときは料金を気にすることはありません。
無償です。改変も、無保証だと明記すれば自由です。

3. サポートは有償です。
問い合わせには、GitHubアカウントに表示しているメールアドレスから
「件名：UKEditorを商用で使いたいのでサポートをお願いしたい。」といれてください。

4. GitHubでスポンサーを募集しています。
開発を支援してくだされば、よりよく効率的に開発ができます。どうぞ、ご検討くださいませ。
```
